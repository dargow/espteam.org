Cube 2: Sauerbraten

Open Source FPS with a built-in editor

Cube 2: Sauerbraten is a free multiplayer & singleplayer first person shooter, the successor of the Cube FPS. Much like the original Cube, the aim of this game is fun, old school deathmatch gameplay and also to allow map/geometry editing to be done cooperatively in-game. Its code is open-source and the game can be downloaded for free on Sourceforge with pre-compiled binaries included for different platforms including Windows, Linux or MacOS X.
